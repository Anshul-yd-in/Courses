# What is a Mobile App?

A mobile app is a software application designed to run on mobile devices such as smartphones and tablets.

## Key Characteristics
- Runs on a mobile operating system like iOS or Android
- Designed for touch-based interaction
- Can access device hardware such as camera, GPS, and sensors
